## 🚨 IF SOMETHING GOES WRONG

### Common Issues & Fixes

| Issue | Solution |
|-------|----------|
| Docker not running | Start Docker Desktop from Start menu |
| Port already in use | Kill process: `taskkill /PID <pid> /F` |
| Build fails | Clear cache: `npm cache clean --force && npm install` |
| Database error | Check DB running: `docker ps` |
| 404 errors | Use `/ai/planner` not old route |
| CORS errors | Check .env `CORS_ORIGINS` |
| Slow response | Check resource usage: `docker stats` |

**See `PLANNER_QUICK_START.md` for detailed troubleshooting.**
