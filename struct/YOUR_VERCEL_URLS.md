## 🚨 IF URLS DON'T WORK

### Frontend URL Issues:
```
Issue: "Cannot reach page"
Fix: 
  ✓ Wait 30 seconds (first deploy is slower)
  ✓ Refresh page (Ctrl+R)
  ✓ Check Vercel dashboard for errors
  ✓ Verify VITE_API_URL environment variable
```

### Backend URL Issues:
```
Issue: "Cannot reach page" or "500 error"
Fix:
  ✓ Add /docs to test if backend runs
  ✓ Check Vercel logs for errors
  ✓ Verify DATABASE_URL is set
  ✓ Verify SECRET_KEY is set
```

### Database Issues:
```
Issue: "Cannot save data" or "500 error"
Fix:
  ✓ Check DATABASE_URL in env vars
  ✓ Make sure ?sslmode=require is included
  ✓ Verify connection string from Neon
  ✓ Check Neon console for quota exceeded
```
