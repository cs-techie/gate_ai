## 🔍 URL TROUBLESHOOTING

### Frontend URL not working
```
✓ Check URL spelling
✓ Make sure deployment finished (green checkmark)
✓ Wait 30 seconds and refresh
✓ Check for CORS errors in console (F12)
```

### Backend URL returns error
```
✓ Check URL spelling
✓ Add /docs to verify API is running
✓ Check environment variables are set
✓ Check Vercel logs for errors
```

### /docs not showing
```
✓ Make sure backend deployed correctly
✓ Check DATABASE_URL is set
✓ Check SECRET_KEY is set
✓ Verify Python version (3.11)
```
