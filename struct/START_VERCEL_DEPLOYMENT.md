```
Go to https://vercel.com
→ Import your gate_ai repository
→ Root directory: packages/frontend
→ Click Deploy
→ Wait 2-3 minutes
→ You get: https://gatexpress-frontend.vercel.app
```

### Step 3: Deploy Backend (15 min)
```
Go to https://vercel.com
→ New Project (same repository)
→ Root directory: packages/backend
→ Add Environment Variables:
   • DATABASE_URL (from step 1)
   • SECRET_KEY (generate via Python)
   • FRONTEND_URL
   • API_URL
→ Click Deploy
→ Wait 3-5 minutes
→ You get: https://gatexpress-api.vercel.app
```

## 🆘 IF YOU GET STUCK

### Common Issues:

**"CORS Error"**
- Check VITE_API_URL in frontend env vars
- See troubleshooting in VERCEL_DEPLOYMENT_COMPLETE.md

**"API not found (404)"**
- Verify backend deployed successfully
- Check Vercel logs for errors
- See troubleshooting section

**"Database connection failed"**
- Verify DATABASE_URL is correct
- Add ?sslmode=require to connection string
- See troubleshooting section

**"Build timeout"**
- Normal on free tier for first build
- Subsequent builds are faster
- Wait for green checkmark

### Need Help?
- Read the troubleshooting section in VERCEL_DEPLOYMENT_COMPLETE.md
- Check official docs: https://vercel.com/docs
- Visit Neon docs: https://neon.tech/docs
